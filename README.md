# 健身房管理系统 java35

## 项目概述

本项目是一款基于Java和SpringBoot开发的健身房管理系统，采用前后端分离的架构设计，前端使用Vue框架。系统支持B/S架构和MVC模式，提供完整的源码及注释，确保代码的透明性和可维护性。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**: 
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术栈**: 
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 系统角色

- **管理员**:
  - 会员管理
  - 健身房管理（包括课程管理、器材管理）
  - 员工管理
  - 报名表管理

- **会员**:
  - 课程查询
  - 在线充值
  - 充值记录
  - Vip管理
  - 购买记录
  - 我的资料
  - 修改密码

## 运行环境

- 服务器：任何支持Java和MySQL的服务器
- 客户端：任何支持现代Web浏览器的操作系统

## 部署步骤

由于部署步骤涉及具体环境配置，本项目提供相关技术栈文档和部署视频指导。

1. 环境准备：确保服务器环境满足开发环境要求。
2. 数据库配置：配置MySQL数据库，导入初始数据。
3. 后端部署：将编译后的后端代码部署到服务器。
4. 前端部署：将编译后的前端代码部署到Web服务器。
5. 启动服务：分别启动后端和前端服务。
